🔒 Politique de confidentialité

Dernière mise à jour : janvier 2025

En résumé : Nous collectons uniquement des données anonymes de navigation (pages visitées, type d'appareil) pour améliorer le site. Nous ne vendons jamais vos données. Nous n'utilisons aucun cookie de tracking publicitaire.

1. Responsable du traitement

Institut MiraLocks
Agoè Cacaveli, Lomé, Togo
Contact : +228 97 98 90 01

2. Données collectées automatiquement

Lors de votre visite sur notre site, nous collectons automatiquement les informations suivantes à des fins statistiques :

  • La page visitée (ex : /gallery.html)
  • La date et l'heure de la visite
  • Le type d'appareil (mobile, tablette, bureau)
  • Le navigateur utilisé (Chrome, Firefox, Safari…)
  • Le système d'exploitation (Android, iOS, Windows…)
  • La résolution d'écran
  • La page d'origine (si vous venez d'un autre site)

Ces données sont anonymes : elles ne permettent pas de vous identifier personnellement.

3. Données collectées via les formulaires

Lorsque vous utilisez notre formulaire de contact ou de rendez-vous, les informations saisies (nom, téléphone, message) sont transmises directement via WhatsApp et ne sont pas stockées sur notre serveur.

4. Finalité du traitement

Les données de navigation collectées sont utilisées exclusivement pour :

  • Comprendre comment les visiteurs utilisent le site
  • Améliorer l'expérience utilisateur
  • Optimiser les performances du site

5. Stockage des données

Les statistiques de visite sont stockées de manière sécurisée via Supabase, un service de base de données conforme aux normes de sécurité internationales. Les données sont conservées pendant une durée maximale de 12 mois.

6. Partage des données

Nous ne vendons, ne louons et ne partageons jamais vos données personnelles avec des tiers à des fins commerciales.

Les seuls accès tiers sont :

  • Supabase : hébergement de la base de données
  • WhatsApp : transmission de vos messages de contact (vous êtes redirigé vers leur plateforme)
  • Google Maps : affichage de la carte sur la page contact

7. Vos droits

Conformément aux lois applicables, vous disposez des droits suivants :

  • Droit d'accès à vos données
  • Droit de rectification
  • Droit de suppression
  • Droit d'opposition au traitement

Pour exercer ces droits, contactez-nous via WhatsApp au +228 97 98 90 01.

8. Cookies

Ce site utilise uniquement des cookies techniques essentiels (stockage local du thème de couleurs choisi). Aucun cookie publicitaire ou de tracking externe n'est utilisé.

9. Modifications

Nous nous réservons le droit de modifier cette politique de confidentialité. Toute modification sera publiée sur cette page avec une date de mise à jour.
